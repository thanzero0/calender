# Web Calendar 📅

A sleek and intuitive calendar application built for the web. Keep track of dates and plan your schedule with this minimalist calendar interface.

## 🌟 Features
- Clean and modern grid layout
- Dynamic date rendering
- Responsive and accessible UI

## 🛠️ Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

## 🚀 Getting Started
Open the `index.html` file in your browser to view the calendar. No setup or server required.
